# Hospitality Revenue & Performance Analytics Dashboard

This project analyzes hospitality business performance using an interactive dashboard built in Power BI. The dashboard tracks key hotel performance metrics including revenue, occupancy, and booking platform contribution to understand operational trends across hotel properties.

---

## Project Overview

The objective of this project is to analyze hospitality KPIs and visualize performance across multiple dimensions such as:

- Hotel properties
- Cities
- Booking platforms
- Room categories
- Time trends

The dashboard enables interactive exploration of hotel performance metrics to identify patterns in revenue generation and occupancy.

---

## Key Metrics Analyzed

The dashboard focuses on industry-standard hospitality KPIs:

- **Revenue**
- **RevPAR (Revenue per Available Room)**
- **ADR (Average Daily Rate)**
- **Occupancy Rate**
- **Realisation %**



## Tools & Technologies

- **Power BI**
- **Power Query**
- **DAX**



## Dashboard Features

- KPI performance cards for hospitality metrics
- Property-level performance comparison
- Booking platform contribution analysis
- Interactive filters and drill-down capabilities
- Revenue and occupancy trend analysis
